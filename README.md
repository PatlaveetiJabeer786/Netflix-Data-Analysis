# 🎬 Netflix Data Analysis Project

A comprehensive data analysis of Netflix's content library using **Python**, **Pandas**, and **Seaborn**.

---

## 🚀 The STAR Overview

### 📍 Situation
Explored Netflix’s global catalog to uncover trends in content additions, regional distributions, and director contributions up to 2021.

### 🎯 Task
Perform **Exploratory Data Analysis (EDA)** to answer business questions regarding peak release years, content types (Movies vs. TV Shows), and data integrity.

### 🛠 Action
* **Data Cleaning:** Removed duplicates and handled missing values using `isnull()` and `dropna()`.
* **Feature Engineering:** Converted date strings to `datetime` objects for time-series analysis.
* **Complex Querying:** Used multi-condition logic (AND/OR) to filter specific datasets (e.g., R-rated movies in Canada).
* **Visualization:** Developed heatmaps for data density and bar charts for categorical comparisons.

### 📊 Result
* **Identified 2019** as the peak year for Netflix content releases.
* **Cleaned ~7,700 rows** of raw data into a reliable analytical format.
* **Top Contributors:** Mapped the top 10 directors and countries driving Netflix content.
